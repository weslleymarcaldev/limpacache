# Icon Instructions

The marketplace requires a `icon.png` file (128x128 pixels) at `media/icon.png`.

## Suggested Design
- Background: dark (#1a1a2e or gradient)
- Foreground: a broom or trash/sparkle icon in purple/white
- The color palette: #a78bfa (purple) on #1a1a2e (dark)

## Tools to create it
- Figma, Canva, or any image editor
- Use the SVG in `sidebar-icon.svg` as a base

## Temporary
Until you add the real icon, remove or comment out the `"icon"` field in `package.json`
to avoid build errors.
